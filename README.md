A akka-http full stack application. (akka, akka-http, slick, bulma, mithril)

## What is it?
A simple TODO management project (Nothing special at all)

## Architecture
* akka (backend)
* akka-http (web-backend)
* slick (db-mangement)
* bulma (design)
* mithril (frontend / js-client)
